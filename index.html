祝所有为实现自己的想法而行动的人收获梦想

发表于 2017-06-21 | 分类于 Life

洛雨：毕业了吗？
林谷：嗯！
洛雨：工作呢？
林谷：有一份不错的工作机会，老板和同事都很nice。但是，自己提不起干劲。
洛雨：哦？所以你想重新找机会？
林谷：是的。
洛雨：有什么想法？
林谷：想去做机器学习。我有编程基础，只要是自己感兴趣的，工作起来更有干劲。
洛雨：你觉得找到自己感兴趣的工作的机率有多大？
林谷：不去做，永远都不知道。
洛雨：如果找不到，你会不会后悔放弃现在的工作机会？
林谷：与其后悔想去做而没去做，不如去做了再后悔。
洛雨：再强调问一次，你真的对机器学习感兴趣？会不会是因为时下机器学习这个概念炒得太热？
林谷：我之前想去搞嵌入式，可是我没去，我失去一次体验嵌入式开发的机会。所以，这次我不想放弃自己的想法。
洛雨：哦？也就是说你要任性一会，以后机器学习的工作也可能让你提不起干劲,你知道如此还坚持自己的想法?
林谷：是的。我想去疯狂一次，去了解自己。
洛雨：既然如此，希望你能够找到自己的兴趣所在。
(陌生人)：祝所有为实现自己的想法而行动的人收获梦想！

机器学习 1

发表于 2017-06-13 | 分类于 Technique

“工欲善其事，必先利其器。机器学习很火，学习机器学习的想法很早就萌生，却苦于无从下手。最近，有幸得到好友—-高手兄指点，念头稍微通达，练习了一些，得到了些许感悟。要搞机器学习，先要知道什么是机器学习，为什么要学机器学习，机器学习学什么，机器学习怎么学，机器学习与数据分析的区别…”

Step 1: 机器学习理论

机器学习理论视频：http://www.imooc.com/learn/717

标注:这个视频将解决我们一开始提出的那些问题。

视频最后一章的Demo源码：source code

Step 2: Python 编程能力

从Step 1 的视频，我们大概了解了机器学习，接下来我们要提高Python 编程能力,由此会带来一个问题：我们学Python要学到什么程度才能搞机器学习？Python是我们学习机器学习的编程工具，既然是编程工具当然是能力越高越好，不过，我们大可不必花太多时间去学习Python。我们的目标是学习机器学习，而不是Python，本末倒置的事情我们不干。 因此，对于Python我们只要懂得基本的语法，以后学习用到的时候，通过查看官方文档等可以解决遇到的问题。
