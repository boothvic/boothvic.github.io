我喜欢安静的夜

发表于 2018-04-14 | 分类于 Life

“我喜欢安静的夜，特别是下雨的夜晚，呆在窗前的书桌，听雨打窗台，或打开网易云音乐的推荐，泡一杯茶，记录生活。”

与朋友闲逛的夜晚，时间总是过得很快。

我喜欢热闹，与朋友一起，下班后转转，买点小吃，来一杯饮料冰沙，是散步，是闲聊，这样的时间很不错。我喜欢安静，放着音乐，泡一杯茶，看一本书，写一些心情。

那就叫随笔吧

发表于 2018-03-24 | 分类于 Life

“人与人的牵绊虽然重要，但孤独也同样重要。”

今天是我公历的生日，但一直以来都是根据家乡的习俗过农历的生日，所以今天也是一个普通的日子。当然，即使今天一如既往，收到兄弟朋友们的祝福还是挺开心的。

清明临近，祭祖时期。林氏家族组织的祭祖活动？行为？算了，就叫祭祀吧。一直以来在外学习的我，家族里的一切都是老爸处理的，今年也不例外。我，有很多疑惑要去解决，有很多想做的事，有很多未看的书……处理事情还不能游刃有余的我，现在只能专心做好眼前的事，所以，别的，以后再说。

问过朋友们一个问题：当一个人意识到自身存在问题，如果改变也是一种逃避，肯定现在以及过去还是改变，Ta该怎么选择？朋友们的回答有些出入，但都表达了一个意思，具体问题具体分析，该变则变。这个问题，两个选择只是为了表明态度，如果说肯定现在以及过去是逃避问题不把问题当问题，那么做出改变是正视问题解决问题。绝大多数人都会选择后者，因为不做出改变，什么都创造不出来，什么都得不到，什么都给予不了。但是，我对这种近似标准答案的选择感到有些恐惧。我希望能够遇到一个毫不犹豫选择肯定现在以及过去的人，或是一个毫不犹豫选择改变的人。比起后者，我更希望遇到前者，因为那样会比较有趣。

Arch 开启pacman和yaourt多线程下载

发表于 2018-03-10 | 分类于 Technique

配置pacman

要修改的配置文件：/etc/pacman.conf，需要root权限，需要下载axel

找到GENERAL OPTIONS，在options下的加上一行：
XferCommand = /usr/bin/axel -n 4 -a -o %o %u
其中各项含义：
-n 4：同时打开４线程
-a　：打印进度信息

配置yaourt

要修改的配置文件：/etc/makepkg.conf，需要root权限，需要下载aria2

将SOURCE ACQUISITION下的DLAGETS的’http::/usr/bin/curl -fLC - –retry 3 –retry-dela y 3 -o %o %u’改成’http::/usr/bin/aria2c -s 4 -m 2 -o %o %u’
-s 4：链接的服务器数量
-m 2：线程数量

拖延，我无法摆脱的噩梦

发表于 2017-12-02 | 分类于 Life

“有一个夜晚我烧毁了所有的记忆，从此我的梦就透明了；有一个早晨我扔掉了所有的昨天，从此我的脚步就轻盈了。”

每天都是新的开始。很久之前就想做出改变，做一个阳光朝气的年轻人。比如早睡早起，清晨跑跑步看看书什么的，晚上跟朋友联系联系，写点东西不要熬夜什么的。可现实恰恰相反，看来我不适合做现充。

昨晚约了朋友今天一起弄几个算法的，鸽了。今天起来太晚，昨晚睡时不注意有些感冒，昏昏沉沉的感觉好不爽。下午４点多才起来，本来以为出去散散步感觉能好一点，我还是太年轻了，回来时感觉有点糟糕。在楼下吃了些东西，尝不出什么味道，买了几个橙子。现在，窗外已是黑夜，在笔记本旁倒上一杯可乐，一直以来感冒时都是这么干的。敲些不知所云的东西，只希望关心我的朋友偶尔能了解到我的最近情况。之所以关闭了空间朋友圈不玩微博，一来我每天没什么变化的状态没什么好跟朋友分享的，二来其中有各种广告，三来沉迷于自己的世界对外界毫不关心。

据说21天可以养成一个习惯。想改变自己每天熬夜不干正事的糟糕现实，挣扎过好几次都没有成效，还任重道远。之前的生活真是没有营养，每天一回来就看看动漫小说……没什么收获，感觉大好时光浪费了，总想做出些改变，但都因为这样那样的借口拖住。早晨的闹钟昨晚定的是７点，闹钟响了之后改为７点半，然后８点，最后拖到上班快迟到了才起，呵呵，我真会折腾闹钟。

小时候挺爱看书的，一直有自己写一本的冲动，之前的情况就像狗咬乌龟－－无从下口。最近生活稍微安定，周六周日有些时间写点东西。不知道要写些什么，总之先写些东西出来，下一步应该可以写些小故事什么的。写作的目的赚钱什么的没有想过，只是希望亲人朋友们在浏览我的博客时能看到我写的东西，偶尔跟我讨论讨论我就很开心了。

给想要努力的自己来些回忆

发表于 2017-11-26 | 分类于 Life

“有能力爱自己，有余力爱别人”

十几年前夏夜，月明星稀，平凡村落里普通的老屋前，如手指般高矮不齐的孩子们，慈眉善目的老人，或讲着故事或让孩子们猜谜。孩子们时而被故事吸引，从中得到激励；时而热情于猜谜，比比谁最先猜到谜底。因为谜底只有一个，是谁最先猜到谁就是第一。得到第一的孩子不跑几圈或不大呼小叫几下是不能安静下来的。老人不厌其烦讲着故事或出谜语，孩子安静的听或者猜和讨论，很温馨的氛围。这时，来了不速之客，孩子们夏天的宠物－－萤火虫们，它们时明时暗的由远及近飞来，孩子们已经按奈不住开始找捕捉器具，之前的安静的气氛一扫而空，或准备扇子或去拿透明的瓶子……孩子们的童年总是欢乐。
我是那群孩子们中的一员，给我们讲故事的那个老人是我奶奶。我很庆幸我有一个这么慈爱的奶奶。我的童年是在伙伴们和老人们的陪伴中度过的。她讲的故事和谜语给我的童年上了七彩，教我为人处世的道理。故事可以说是引经据典，三字经，诗经……这些古典书籍上的故事，受此影响我能静下心来看古典书籍，初中时因此买书看了论语诸子百家诗经。七岁上学第一天，她特意给我做了葱汤，喝了葱汤读书更聪明的蕴意，给了我“七岁应神童，天生我姓聪”的美好祝福，后来我也算争气。
她的故事中的女主都是诗经关雎中的窈窕淑女，小时候对窈窕淑女毫无概念，无法具象化，只知道是个漂亮的人。长大了才知道，窈窕淑女或是大家闺秀或是小家碧玉，或是……男人有多少个她就有多少种，色气的大姐姐，邻家羞涩的小妹妹，亦或是从天儿降的傲娇……对我来说，窈窕淑女必定是知书达理的。

这些年遇到过很多可爱的女生，我一直认为，她们是幸福的化身，跟她在一起的人一定能得到幸福。一个不喜欢自己的人我不知道有什么资格去追别人，或许把喜欢放在心底先让自己强大是一种不错的选择。

姑妄言之，姑妄听之

发表于 2017-11-25 | 分类于 Life

“有贵人相助”

朋友最近很不开心，六神无主之下跑去问仙。
问：我什么时候才能赚大钱？
答：成家，有了家庭就有。
问：我和前女朋友还有复合的可能吗？
答：有音信来就有。
问：我是待在Ｓ城好还是去Ｙ城发展好？
答：都好，城里都好。
朋友一阵沉默，伸手摸了摸口袋的香烟又放开。
老神仙慈眉善目：年轻人，靠不了父辈祖辈，就靠自己，你不久就会得到贵人相助。
朋友留下了今天买烟的钱后转身离开。
(陌生的声音：我好像在哪听过这些话。)

人处于低谷的那段时间，那种深刻的无助感令人智息，你没有看错，是智息。这段时间理性往往会被感性取代，为了尽快逃离，自甘堕落也是一种解脱。

下一站

发表于 2017-11-18 | 分类于 Life

“择一城终老，遇一人白首”

一直以来，时常有写一些什么的冲动，每当拿起笔或者触摸键盘时，却不知道要写些什么。
我不是文人，没有多少文采，甚至看过的故事放开书后自己也讲不清楚。朋友说我这是缺少自信，劝我要自信自己讲的故事比书上描述的还精彩。每当这时我往往用右手稍微挠着后脑勺，嘴角微微翘起，然后是沉默。朋友这时已经找到了别的话题。
我看过很多动漫，日本的、国内的……只要觉得有趣或有意思就看，就像没有人会因为笑话好笑而不看下去。现在很少看了，偶尔才有觉得有趣的，跟看笑话使得笑点渐渐变高了一样。小说和漫画也看过一些，相对动漫来说就少的多了。当然这里的动漫就不要纠结是动画和漫画这些细节，全当是动画了。有时候手机滑动了一圈也不知道要点开哪个APP，手机厌烦症？应该是太闲了。这时我初中那会念过的论语就派上用场，脑海里出现一组奇怪的画面，一个十五六岁的年轻人，整天吃饱了就睡，偶尔坐起来目光呆滞的看着天或者灰茫茫的远方。然后一个老者出现在年轻人面前，口中念念有词，“饱食终日，无所用心，难矣哉！不有博弈者乎，为之，犹贤乎已”，最后年轻人左手拿书右手执棋坐在棋盘前，天空一片湛蓝，远方的行人隐约可见。
